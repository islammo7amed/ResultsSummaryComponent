# ResultsSummaryComponent
A **responsive "Results Summary" component** built using **HTML and CSS** as part of a Frontend Mentor challenge. This component displays a user's test results along with a summary of individual skill scores, using a modern, colorful design inspired by gradients and minimalism.

## Features
- Fully **responsive layout** optimized for mobile and desktop screens.
- **Gradient backgrounds** and rounded elements for a modern UI.
- Styled with **custom font** (`Hanken Grotesk`) for clean typography.
- Interactive-looking **Continue button** with gradient hover effect.
- Semantic HTML structure with accessibility in mind.

## Technologies Used
- HTML5
- CSS3 (Flexbox, Media Queries)
- Custom Fonts
- CSS Gradients
